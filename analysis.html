Discussion on Algorithms

Ford Fulkerson

In Ford Fulkerson we run algorithm in 3 broad steps i.e, first we assume the initial flow to be zero, next we check for augment path from source to sink and add this path's flow to overall flow, and lastly we return flow when there are no augment path in existance.


Bipartite Problem

For Bipartite Problem we ran the input through same ford fulkerson algorithm with following change: We assumed source to be zero and all vertices of N-set and added by no.of vertices in M-set and sink is set to be no.of vertices in M-set + N-set + 1.


Timing Analysis

Time complexity of the Ford Fulkerson algorithm is O(max_flow * no_of_Edges). The time taken by small graphs (under 100 nodes and 300 edges) was always under 0.008sec of which the highest being 0.005343 sec for small_g1.txt .
